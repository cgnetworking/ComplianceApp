Ubuntu prerequisite install

`scripts/local_setup.sh` runs these Ubuntu steps automatically. Source: Install PowerShell 7 on Ubuntu 

sudo apt-get update
sudo apt-get install -y wget apt-transport-https software-properties-common
source /etc/os-release
wget -q https://packages.microsoft.com/config/ubuntu/$VERSION_ID/packages-microsoft-prod.deb
sudo dpkg -i packages-microsoft-prod.deb
rm packages-microsoft-prod.deb
sudo apt-get update
sudo apt-get install -y powershell

Run Microsoft Zero Trust Assessment with certificate auth

The PowerShell module is installed by `scripts/local_setup.sh`. Source: Get started with the Zero Trust Assessment 

pwsh
Connect-ZtAssessment -ClientId YOUR_APP_ID -TenantId YOUR_TENANT_ID -Certificate YOUR_CERT_SUBJECT
Invoke-ZtAssessment -Path /path/to/output
